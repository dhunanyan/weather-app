# 🌤️ Weather App

Welcome to the **React Native Weather App**! This mobile-friendly app provides current weather conditions with a user-friendly interface, built using React Native and Expo Go. Whether you’re checking the weather in your city or planning for a trip, this app provides quick, reliable, and up-to-date weather information.

## 📖 About

This app is designed with React Native, making it compatible with both iOS and Android. Using Expo Go simplifies testing and deployment, so you can quickly access all of the app’s features without hassle. The weather data is sourced from a reliable API, ensuring accuracy and consistency.

## ✨ Features

- **Real-time Weather Data**: Get updated weather details for any location.
- **Location-based Search**: Find weather conditions for your current location or search other areas.
- **Intuitive UI**: Designed with ease of use in mind, ensuring smooth navigation and readability.
- **Cross-platform Compatibility**: Works on both iOS and Android via Expo Go.
